# Recursos - Imagen de Fondo del Hero

## Imagen requerida

Coloca aquí la imagen que se usará como **fondo del hero section** en la página de Recursos.

### Nombre del archivo
`resources.png`

### Especificaciones recomendadas
- **Formato**: PNG o JPG
- **Tamańo**: Mínimo 1920x600 px (Full HD ancho)
- **Estilo**: Imagen de líderes, reunión, o equipo de iglesia
- **Contenido**: La imagen se mostrará con un overlay azul oscuro semitransparente para mantener la legibilidad del texto blanco

### Ubicación
`wwwroot/img/resources/resources.png`

---

Una vez colocada la imagen, se mostrará automáticamente como fondo del hero section con un overlay azul que permite ver parte de la imagen mientras mantiene el texto legible.

### Comportamiento
- La imagen cubre todo el ancho del hero section
- Se aplica un overlay azul oscuro (85% de opacidad) sobre la imagen
- El texto blanco se mantiene legible sobre el fondo
- Es completamente responsive
